# Recommended Practice (RP-023) Renaming of CC91 and CC93

## Page 1

Recommended Practice (RP-023)
Renaming of CC91 and CC93
Due to the fact that so many manufacturers use Control Change #91 and #93 in the same way, they should be renamed
according to their common usage.
Details: Sets Control Function names as follows:
CC# (hex) 	[Old Name] 	[New Name]
===================================
91 (5BH) 	[Effect 1 Depth] 	Reverb Send Level
93 (5DH) 	[Effect 3 Depth] 	Chorus Send Level
The actual response is dependent on a formal R/P such as General MIDI Level 2 Recommended Practice.
