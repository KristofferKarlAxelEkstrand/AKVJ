# Goal: The Ultimate MIDI MCP Server

The primary goal of the MIDI MCP Developer workflow is to build a **great Model Context Protocol (MCP) server that makes working with MIDI easier**. 

This server must empower AI coding agents (and humans) to easily check, verify, and utilize the official MIDI specifications (MIDI 1.0, MIDI 2.0, Web MIDI API, etc.) without having to hunt through dense PDFs or fragmented documentation. 

Every transformation, schema design, and new tool should be evaluated against this core objective: **Does this make the MIDI specification more accessible, queryable, and usable for building MIDI applications?**
