# Metadata Edit Form Improvements

The metadata edit form in the mainframe UI could use some improvements based on previous notes.
Please add the following:
- A validation check for the `png` filename existence when editing metadata. If the user changes it to a file that doesn't exist, it should warn them.
- A clip sort dropdown in the Library tab to help organize large clip buckets (e.g., sort by name, size, role).
- Consider adding support for batch operations if it fits naturally.
